* Each transaction carried out in the system must ONLY contain the following information:
   1) Transaction Date (Date)
   2) Transaction Type (Deposit, Transfer, Bill Payment)
   3) Transaction Value (value greater than zero)

* Whenever the transaction is of the DEPOSIT type, the transaction amount must be added to the account balance.
* Whenever the transaction is of the TRANSFER or BILL PAYMENT type, the transaction amount must be subtracted from the account balance.
* The balance must always be updated on the application screen with each transaction carried out.


* Cada transação realizada no sistema deve possuir SOMENTE as seguintes informações:
  1) Data da Transação (Date)
  2) Tipo de Transação (Depósito, Transferência, Pagamento de Boleto)
  3) Valor da Transação (valor maior que zero)

* Sempre que a transação for do tipo DEPÓSITO, o valor da transação deve ser adicionado ao saldo da conta.
* Sempre que a transação for do tipo TRANSFERÊNCIA ou PAGAMENTO DE BOLETO, o valor da transação deve ser subtraído do saldo da conta.
* O saldo deve sempre ser atualizado na tela da aplicação a cada transação realizada.
